🤖 Image to 3D Relief Bot

Our Telegram bot @Image_2_3D_Bot converts any image to a high-quality grayscale depth map for CNC.

  • ✅ 99% accuracy – preserves fine details, text, faces
  • ✅ Automatic background removal & straightening
  • ✅ Pay per use (diamonds) – no subscription

How to use the bot

1. Start the bot → /start
2. Buy diamonds via UPI QR (10 💎 = ₹30)
3. Send a clear design image → get relief map in 30-60s
